What is esphomelib?Â¶

Esphomelib is a framework for creating custom firmwares for your WiFi-enabled ESP microcontrollers. Its primary focus is making the process to get your ESP running as simple as possible, with many helper tools to ensure you will have the best user experience.

Esphomelib is split up into two main parts: esphomelib, the C++ framework backing the entire framework, and esphomeyaml, a tool that automatically creates firmwares for you just from simple configuration files - so no programming experience required!
